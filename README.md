# Founder — Sistema de Gestão do Projeto

Este repositório organiza o desenvolvimento do produto, o aprendizado técnico e o uso de IA como parceira de jornada.

## Estrutura

| Pasta | O que contém |
|---|---|
| `/produto` | Contexto, decisões e roadmap do produto |
| `/tecnico` | Documentação técnica por área |
| `/jornada` | Currículo vivo e habilidades validadas |
| `/governanca-ia` | Como a IA é usada, critérios e prompts |
